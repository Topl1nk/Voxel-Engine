# Voxel Engine - Phase 5

## Запуск проекта

**ВАЖНО:** Этот проект использует ES6 модули, поэтому его нельзя открывать напрямую через `file://` протокол. Необходимо использовать локальный веб-сервер.

### Способ 1: Python HTTP Server

```bash
python -m http.server 8000
```

Затем откройте в браузере: `http://localhost:8000`

### Способ 2: Node.js (если установлен)

```bash
npx http-server -p 8000
```

### Способ 3: VS Code Live Server

Если используете VS Code, установите расширение "Live Server" и откройте `index.html` через него.

## Управление

- **WASD** - движение
- **SPACE** - прыжок
- **1-7** - выбор блока
- **ЛКМ** - разрушить блок
- **ПКМ** - поставить блок
- **ESC** - выйти из режима pointer lock (пауза)

```
├── index.html          # Основной HTML файл

│   └── style.css      # Стили
```
